# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting house prices using the Ames Housing dataset, which contains 79 explanatory variables describing various aspects of residential homes.

The objective is to build and compare multiple machine learning models to accurately predict the sale price of houses and analyze the key factors influencing property value.

---

## 🎯 Problem Statement
1. Perform complete data analysis on the dataset.
2. Build robust machine learning models to predict house prices.
3. Determine relationships between house features and price.
4. Provide suggestions for customers based on price-driving factors.

---

## 📊 Exploratory Data Analysis (EDA)
- Analyzed distribution of SalePrice
- Checked correlation between features
- Identified top price-influencing variables
- Visualized relationships using scatter plots and heatmaps

---

## 🛠️ Feature Engineering
- Handled missing values
- Converted categorical variables using encoding techniques
- Selected important features based on correlation and model importance

---

## 🤖 Machine Learning Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

---

## 🏆 Model Comparison

| Model                | R² Score |
|----------------------|----------|
| Linear Regression    | ~0.80+   |
| Random Forest        | ~0.88+   |
| Gradient Boosting    | ~0.90+   |

Gradient Boosting achieved the highest R² score and was selected as the best performing model.

---

## 🔍 Key Insights
- Overall Quality strongly impacts house price.
- Above-ground living area significantly increases valuation.
- Garage capacity positively influences price.
- Newer houses tend to have higher sale prices.
- Location (Neighborhood) plays a major role in pricing.
